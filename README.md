Existing API Endpoints
======================
1) List Locations:
   **[GET]** `http://localhost:8088/api/locations`

2) Fetch Location Details:
   **[GET]** `http://localhost:8088/api/locations/{latitude,longitude}`

3) Add Location:
   **[POST]** `http://localhost:8088/api/locations`
